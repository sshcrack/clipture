# <a href="#">Download & Install</a>

## Selfhosting
Make sure you have yarn installed, if not sure:
```bash
npm i -g yarn
```

Clone repository
```bash
git clone https://github.com/sshcrack/clipture-server
```
Replace defaults with your config files in .env.local

Build:
```bash
 yarn build
```

Start: 
```bash
  yarn start -p 3001
```
