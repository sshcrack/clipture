## Selfhosting
Make sure you have yarn installed, if not sure:
```bash
npm i -g yarn
```

1. [Setup server](https://github.com/sshcrack/clipture-server)
2. Clone Repo
3. Update variables in src/Globals/
4. Install
 ```bash
  yarn
 ```

### Releasing
Generate setup files:
```bash
yarn make
```
Your files will be available in out/make
